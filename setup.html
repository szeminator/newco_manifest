03 — Die kritische Lücke

Der Representative — die wichtigste Einstellung.

Lorenz öffnet die Tür. Hannes baut die Lösung. Aber wer sitzt dazwischen — beim Fachbereich, beim Kunden, beim wöchentlichen Status?

Ohne diese Person ist Hannes in jedem Kundengespräch, jedem Scope-Workshop, jedem Projektupdate. Das ist kein CTO-Job. Das ist eine eigene Rolle.

Kandidaten: intern aus der Diamir oder extern. Suche startet parallel zur Gründung.

Technisches GrundverständnisVersteht was ein Agent ist, was eine API macht, was n8n tut — muss es nicht bauen können.
Fachbereichs-EbeneKommt gut an bei Abteilungsleitern und Projektverantwortlichen, nicht nur C-Level.
Scope-FähigkeitKann aus einem offenen Gespräch einen klaren, umsetzbaren Projektscope ableiten.
Präsenz-BereitschaftFährt zum Kunden. Wien, Graz, Linz, Zürich. DACH bedeutet Reisen.
Consulting-BackgroundErfahrung in Projektdefinition oder Beratung von Vorteil — kein Muss.
